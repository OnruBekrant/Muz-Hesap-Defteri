# Muz Hesap Defteri

Teknolojiyle arası çok iyi olmayan çiftçiler için basit, büyük butonlu ve sade arayüzlü hesap defteri.

## Kurulum

1. Bağımlılıkları yükleyin:
   ```bash
   flutter pub get
   ```

2. Hive adaptörlerini oluşturun:
   ```bash
   flutter pub run build_runner build --delete-conflicting-outputs
   ```

3. Uygulamayı çalıştırın:
   ```bash
   flutter run
   ```
